### 개요
* 한국과 일본 간 신호 대기 시간을 비교하고 싶어 만듦
* 교통 혼잡도, 신호 대기시간에 따른 '평균 대기시간' 을 측정함
* 페이즈에 따른 좌회전, 우회전 기능도 구현
* tkinter을 이용한 GUI 구현, `terminal_runner.py` 로 빠른 데이터 추출도 가능
* 보행자와 대각선 횡단보도도 하고 싶었으나 스펙 상 미구현

### 결과
* 데이터 상 일반적인 교통상황에서 신호는 20~30초 대기시간이 가장 효율적임
* 일본은 기억 상 1분 이내, 한국은 100초 정도였던 것으로 기억
* 안전 상 이유, 실제 사람의 반응 등을 고려한 시스템이라고 추측함
